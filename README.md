# Macro Tracker

A simple mobile application built with React Native and TypeScript for tracking daily nutrition intake.

## Features

* Add and manage meals
* Track calories, protein, carbohydrates, and fat
* View nutritional information for each meal
* Local data persistence
* Share meal information
* Copy data to clipboard
* Notifications and reminders
* Responsive mobile UI

## Tech Stack

* React Native
* Expo
* TypeScript

## Purpose

I built this project to gain hands-on experience with React Native and explore its built-in APIs, including notifications, sharing, clipboard integration, and local storage.

## Installation

```bash
npm install
```

## Run the App

```bash
npx expo start
```

Then open the app using Expo Go, an Android emulator, or an iOS simulator.
